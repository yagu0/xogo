# xogo.casa

Simplified version of old vchess.club, to focus on the essential : the game.

## Requirements (dev)

PHP + Node.js + npm.
```npm i -g nodemon```

## Usage

Initialisation (done once):

```./initialize.sh```

You may want to edit the parameters.js file. Then:

```./start.sh``` (and later, ```./stop.sh```)
